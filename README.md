# DesktopGridSnapper

DesktopGridSnapperは、デスクトップ上のウィンドウやアイテムをグリッドにスナップさせ、効率的なデスクトップ管理を実現するツールです。本ツールは、カスタマイズ可能なグリッドおよび設定ウィンドウを提供し、簡単にデスクトップ環境を整えることができます。

## 主な機能

- デスクトップグリッドの表示とカスタマイズ
- ウィンドウやアイテムのグリッドへの自動スナップ
- 設定ウィンドウを通じた柔軟なカスタマイズ
- **-silentオプション**: 起動時にグリッドと設定ウィンドウを表示せず、バックグラウンドで常駐実行

## 使用方法

1. アプリケーションを起動します。
2. グリッドと設定ウィンドウを使用して、デスクトップ環境をカスタマイズします。
3. スナップが有効になっていることを確認してください。

### -silentオプション

```bash
DesktopGridSnapper -silent
```

`-silent`オプションを付けて起動すると、グリッドや設定ウィンドウを非表示にした状態で常駐します。このモードでは、バックグラウンドで動作し、シームレスなデスクトップ操作を提供します。
